# TeleDrivePro
TeleDrivePro is a lightweight unlimited storage solution by Vedhant Khajuria. No login or signup required—just create a small 2–3 MB local backend folder on your device and enjoy seamless unlimited file storage, upload, and access. Fast, simple, and privacy-focused.
